# Система управления задачами с приоритетами

Учебный проект на Python.
Программа позволяет вести список задач с указанием названия, приоритета, времени выполненя и дедлайна. Также поддерживается отмена последних действий, очередь исполнения и быстрый поиск задач по дедлайну при помощи бинарного дерева.

## Возможности

* Добавление задач
* Удаление задач
* Редактирование задач
* Отмена последних действий (Stack)
* Очередь задач на выполнение (Queue)
* Бинарное дерево поиска по дедлайнам (BST)
* Вывод задач по возрастанию дедлайна
* Вывод задач по уровню приоритетности
* Поиск самой ранней задачи
* Поиск самой поздней задачи

## Используемые структуры данных

* List
* Stack
* Queue (deque)
* Binary Search Tree (BST)

## Запуск программы

```bash
python main.py
```
